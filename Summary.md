# Summary of instructions

This file contains a summary of the main paths to introduce different contents to your website.
  
  * Change any details about the Home section and the profile picture [here](https://github.com/pmramiro/starter-academic/tree/master/content/authors/admin)                                                       
  * Change any details about the titles and content of other sections [here](https://github.com/pmramiro/starter-academic/tree/master/content/home)
                                                                             
  * Add custom widgets for the different sections [here](https://github.com/pmramiro/starter-academic/tree/master/layouts/partials/widgets)
  
  * Add any icons you want to show in your website [here](https://github.com/pmramiro/starter-academic/tree/master/assets/media/icons/brands)
                                                                            
  * Add the icon you want to show in your Chrome tab [here](https://github.com/pmramiro/starter-academic/tree/master/assets/media)
                                                          
  * Define website name, url, manus order, contact details and other options [here](https://github.com/pmramiro/starter-academic/tree/master/config/_default)
                                                                                    
  * Add custom fonts and themes [here](https://github.com/pmramiro/starter-academic/tree/master/data)
                                                                                    
  * Add CV or any other documents [here](https://github.com/pmramiro/starter-academic/tree/master/static/uploads)
